This comprehensive file utilities package provides:

1. **Core file and folder operations** (read, write, copy, move, delete)
2. **File information and metadata** (size, dates, permissions)
3. **Advanced search capabilities** (by name, extension, size, content, date)
4. **Compression and archiving** (ZIP, TAR, GZIP)
5. **File validation** (existence, size, type, permissions, safety)
6. **Utility functions** (path operations, backups, duplicates detection)

The package is modular, well-documented, and follows Python best practices. It includes proper error handling, type hints, and comprehensive examples for easy usage.
